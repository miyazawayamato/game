## RPG

### アプリ概要
矢印キーを使い右下のゴール（黄土色のパネル）を目指すゲームです。  
途中の敵にぶつかるとバトルが発生してテキストが表示されます  
味方パーティーのHPが０になるとゲームオーバーになります  
オブジェクト指向学習に利用  
[サイトへ移動](https://dokusika.deca.jp/game/rpg/)

### アプリ機能
map内のキャラを矢印キーで動かす（canvas）  
バトルテキスト  
ajax  


### 使用技術
言語：javascript php  
開発環境：xampp  
デプロイ：レンタルサーバー（ロリポップ）
